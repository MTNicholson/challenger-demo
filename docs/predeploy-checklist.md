- [ ] Используется совместимая LTS-версия Node.js и npm.
- [ ] Зависимости установлены по актуальному `package-lock.json` без добавления новых пакетов.
- [ ] Подтверждено, что переменные окружения текущему frontend-only демо не требуются.
- [ ] Подтверждено, что приложение использует только моковые данные из `src/data` и не зависит от backend, auth, базы данных, платежей, камеры или внешних интеграций.
