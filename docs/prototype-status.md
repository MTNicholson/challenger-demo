# Статус прототипа

## Готово к демо

- Публичный лендинг — готов.
- Пользовательский демо-путь — готов.
- Кабинет бренда — готов.
- Сценарий сканера — готов в моковом виде.
- Аналитика — готова как визуальный мок.
- Документация по развёртыванию — готова.

## Границы текущей версии

Прототип полностью frontend-only и использует только моковые данные из `src/data`. В нём нет backend, авторизации, базы данных, платежей, доступа к камере и реального QR-сканирования. Визуальный дизайн согласован для текущего показа, но более глубокая полировка и систематизация дизайн-системы остаются задачами следующего этапа.
