# Маршруты проекта

Проект использует App Router Next.js. Все маршруты являются frontend-only демо-экранами и работают на моковых данных из `src/data`.

## Публичная зона

- `/` - публичный маркетинговый лендинг: объясняет идею "Челленджера", ценность для пользователей и брендов, ведет к демо-сценариям.

## Пользовательское демо

- `/app` - главная пользовательского приложения: приветствие, баланс монет, активный челлендж, рекомендации и задача дня.
- `/app/challenges` - каталог челленджей: показывает доступные механики, партнеров, награды и условия.
- `/app/challenges/[id]` - детальная страница челленджа: демонстрирует условия, прогресс и награду по конкретному челленджу.
- `/app/active-challenge` - экран активного челленджа: показывает текущий прогресс пользователя в выбранной механике.
- `/app/map` - карта демо-точек: показывает партнерские места и челленджи рядом.
- `/app/profile` - профиль пользователя: демонстрирует персональную информацию, уровень и историю активности.
- `/app/coins` - монеты: показывает баланс, начисления и смысл внутренней валюты.
- `/app/reward` - награда: демонстрирует экран получения или использования награды.

## Кабинет бренда

- `/brand` - обзор кабинета бренда: ключевые метрики, активные механики и быстрый доступ к разделам.
- `/brand/challenges` - челленджи бренда: список запущенных механик и их параметры.
- `/brand/challenges/new` - создание челленджа: демо-форма настройки новой механики.
- `/brand/analytics` - аналитика: демонстрирует метрики вовлечения, активности и наград.
- `/brand/rewards` - награды: управление вознаграждениями для пользователей.
- `/brand/scanner` - сканер сотрудника: демо-экран проверки пользователя или награды.
- `/brand/scan-result` - результат сканирования: показывает состояние после проверки.
- `/brand/preview` - гостевое превью бренда: демонстрирует, как бренд выглядит для пользователя.
